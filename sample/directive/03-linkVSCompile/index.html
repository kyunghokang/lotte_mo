Directive compilation vs linking function

compilation stage는 app 전체 lifecycle에서 한번만 실행되고, scope 생성 전에 실행됨에 따라 scope 변수를 사용할 수 없다.
linking stage는 scope가 directive에 attach 될 때 실행됨에 따라 scope 변수 사용이 가능하다.

* ng-repeate에서 반복적으로 호출되는 directive의 경우 매번 실행이 되므로 복잡한 로직을 처리하는 것은 바람직하지 않다.
compile 레벨에서 element를 한번에 처리하도록 로직의 구성을 하는 것이 필요하다.
